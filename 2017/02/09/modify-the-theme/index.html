为maupassant主题加上google-code-prettify

Maupassant最初是由Cho大神为Typecho平台设计开发的一套响应式模板,非常简洁、清爽、in,由icylogic大神迁移到Hexo版本，经过屠夫9441将相关的功能添加，目前这套主题已经相当完善了。无奈写博客时 代码格式个人不是很喜欢，蛮喜欢stackoverflow的风格，于是开始动手把。

关于google-code-prettify

项目地址: https://code.google.com/p/google-code-prettify/

Google出品的代码高亮JavaScript库， 作者建议采用快速嵌入的方式去实现

#script src="https://cdn.rawgit.com/google/code-prettify/master/loader/run_prettify.js">#/script#

不过在大天朝有墙，访问不了google，所以不推荐用这种方式。

配置

网上有不少文章介绍添加方法，不过大部分添加的方式 作者都使用的是ejs的模板，而Maupassant这套主题用的是Jade的模板。先下载js文件，
node下载

npm install google-code-prettify --save
或者直接在github下载。我是用node直接下载的， 将下载好的prettify.js和prettify.css这两个文件拿出来，放在*/themes/maupassant/source*的js和css文件夹下 然后打开*/themes/maupassant/layout/_partial/after_footer.jade*文件，在文件最后添加引用 
script(type='text/javascript', src=url_for(theme.js) + '/prettify.js' + '?v=' + theme.version)
打开*/themes/maupassant/layout/base.jade*，在 
link(rel='stylesheet', type='text/css', href=url_for(theme.css) + '/style.css' + '?v=' + theme.version)
下面添加引用 
link(rel='stylesheet', type='text/css', href=url_for(theme.css) + '/prettify.css' + '?v=' + theme.version)

OK,大功告成，现在只需要只需要用#pre#和#/pre#将你需要高亮的代码包裹起来就可以了，#是<>尖括号。 这里还有不同的代码样式， 选择喜欢的样式直接保存覆盖css文件即可
