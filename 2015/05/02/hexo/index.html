配置Hexo

之前一直用Octopress博客，Octopress好久没更新了，改用Hexo。 hexo是使用node.js技术开发的一个纯静态博客系统，和Octopress原理一样，都是生成静态文件。

安装Hexo很简单

sudo npm install -g hexo
系统安装了hexo， cd到一个准备安装的目录， 
cd Hexo
hexo init
npm install
执行完成之后会在Hexo文件夹下面有一系列文件 接下来安装主题， 挑了好久的主题，最终选择了maupassant主题 
git clone https://github.com/tufu9441/maupassant-hexo.git themes/maupassant

npm install hexo-renderer-jade --save

npm install hexo-renderer-sass --save

和Octopress一样，在github上配置好对应的repo，然后打开Hexo下面的_config.yml文件，修改

theme: maupassant
deploy:
    type: git
    repo: https://github.com/用户/用户.github.io.git

然后执行： 

npm install hexo-deployer-git --save

新用户可能要配置本机的ssh和github上对应， 然后执行

hexo deploy
至此，搭建完成。 打开 用户名.github.io 应该可以看到你的个人页面了 ## 一些常用的命令 
hexo new"postName" 新建一片文章

hexo new page"pageName" 新建页面

hexo generate 生成静态页面

hexo server 本地服务器

hexo deploy 部署到git

自定义操作

  • 添加评论 theme/…/_config.yml 设置一下duoshuo的shortname就搞定了，具体去多说官网注册一下就可以了。 然后 hexo g -d
  • 添加百度统计 theme/…/_config.yml 设置一下baidu_analytics搞定，具体去百度统计网站注册一下就可以了。 然后 hexo g -d
  • 更新hexo主题 
cd themes/主题

git pull
